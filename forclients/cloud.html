サービス提供について

もっと賢くクラウド環境を使うには

クラウド環境を使った、低コスト、高拡張性システムの構築をサポートいたします。

クラウド環境でコスト削減

Amazon EC2/S3、Google App Engineといったクラウド環境の採用により、低コストでのサイト運営をご提案いたします。
初期コストの削減、拡張性の向上が実現できます。

高効率分散処理をクラウド環境で

バッチ処理の長時間化、処理量の限界を感じていませんか？ Amazon Elastic MapReduceへの移行により、処理時間の短縮、処理データ量の拡大、サーバコストの削減を実現できます。

Amazon Elastic MapReduceとは？

MapReduceとは、Google社によって考案された分散処理を行う仕組みです。Amazon Elastic MapReduceは、Amazon Web Servicesのクラウド環境でMapReduceを実行します。
設定の変更だけで、サーバのスペックや台数を簡単に切り替えることができます。実行時間以外ではサーバコストがかからないため、コスト削減にも寄与します。

実行時間・コスト例

Amazon EC2（連続稼動）のみで行っている一日一回のバッチ処理を、Amazon Elastic MapReduceに置き換えた場合の例です。

システム Amazon EC2のみ Amazon Elastic MapReduce
インスタンスタイプ ハイCPUミディアム ハイCPUミディアム
インスタンス数 1 5
一回の実行時間 約10時間 約3〜5時間（データ入出力を含む）
月あたり価格 $144（約11,261円） $103.5（約8,094円）

【注1】 円換算は2011年11月6日現在のレートによります。【注2】東京リージョンの価格です

事例
【S社様】データ分類処理を国内の専用サーバからElastic MapReduceに移行
1日に一度しか動かない処理のための高スペックサーバを解約し、維持コストを削減することに成功しました。
【K社様】文章からのキーワード抽出処理・データひもづけ処理をAmazon EC2からElastic MapReduceに移行
現実的な実行時間で、処理可能なデータ量を約10倍にすることができました。
